# Glass Calculator

## Description

A beautiful glass-style calculator built using HTML, CSS, and JavaScript. 
It supports both button clicks and keyboard input, and features a modern 
glassmorphism design with responsive layout.

## Features 

    - Glassmorphism UI with blurred background  
    - Responsive layout (mobile-friendly)  
    - Fully functional calculator (Add, Subtract, Multiply, Divide)  
    - Keyboard support (use numbers, operators, Enter, Backspace, Escape)  


## How To use

    1. Clone or download this project folder.
    2. Open `index.html` in any browser (Chrome, Edge, Firefox, etc).
    3. Use mouse clicks or keyboard to perform calculations.
