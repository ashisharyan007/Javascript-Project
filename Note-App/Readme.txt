# NOTES APP

## Description
This is a simple and responsive Notes App built using HTML, CSS, and JavaScript. 
It allows users to create, edit, delete, and store notes in their browser using localStorage. 
Each note is saved even after refreshing the page or closing the window.

## Features

    - Create unlimited notes
    - Edit notes inline
    - Delete individual notes using a trash icon
    - Notes are automatically saved using localStorage
    - Clean UI with a modern gradient background
    - Fully responsive design for desktop and mobile devices

## Technologies Used

    - HTML5
    - CSS3
    - JavaScript (DOM + localStorage)

## How to Use

    1. Open the project folder.
    2. Make sure the following structure is present:

        /project-folder
        ├── index.html
        ├── style.css
        ├── script.js
        └── images
             └── delete.png
             └── edit.png
             └── notes.png

    3. Open `index.html` in a web browser.
    4. Click the "Create Note" button to add a new note.
    5. Type in the note directly.
    6. Click the trash icon to delete a note.

Notes:
------
- Notes are saved automatically in the browser using localStorage.
- Notes will remain even after refreshing or closing the tab.
- The delete icon must be present in the images folder.

## How to Run
No setup required. Simply download or clone the repository and open `index.html` in your web browser.
