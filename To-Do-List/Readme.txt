# To-Do List App

## Description
A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.  
This app allows users to add, delete, and mark tasks as completed.  
One key feature is that the tasks are saved in the browser using localStorage, so even after refreshing or closing the browser, the data remains intact.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Automatically saves data in the browser using localStorage
- Responsive design

## How to Use
1. Open the `index.html` file in any web browser.
2. Enter your task in the input box and click "Add".
3. Use the checkmark to mark a task as done.
4. Use the delete button to remove a task.
5. All tasks are saved automatically in your browser.

## Technologies Used
- HTML
- CSS
- JavaScript
- localStorage API

## How to Run
No setup required. Simply download or clone the repository and open `index.html` in your web browser.
