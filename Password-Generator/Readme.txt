# RANDOM PASSWORD GENERATOR

## Description
This is a simple and responsive Random Password Generator built using HTML, CSS, and JavaScript.  
It allows users to generate secure random passwords and copy them to their clipboard with a single click.  
The app features a clean UI and is optimized for both desktop and mobile devices.

## Features

    - Generate strong random passwords instantly
    - Includes uppercase, lowercase, numbers, and symbols
    - Copy the generated password to clipboard by clicking the copy icon
    - Visual feedback when the password is copied
    - Clean UI with a stylish linear gradient background
    - Fully responsive design for desktop, tablet, and mobile screens

## Technologies Used

    - HTML5
    - CSS3
    - JavaScript (DOM)

## How to Use

    1. Open the project folder.
    2. Ensure the following structure is present:

        /project-folder
        ├── index.html
        ├── style.css
        ├── app.js
        └── images
             └── copy.png
             └── generate.png

    3. Open `index.html` in a web browser.
    4. Click the "Generate Password" button to create a new password.
    5. Click the copy icon to copy the password to your clipboard.
    6. A message will appear confirming the password was copied.

Notes:
------
- Password includes at least one uppercase letter, lowercase letter, number, and symbol.
- Copy confirmation is shown below the input field after clicking the icon.
- Ensure the `copy.png` and `generate.png` images are located in the `images` folder.

## How to Run
No setup required. Simply download or clone the repository and open `index.html` in your web browser.
