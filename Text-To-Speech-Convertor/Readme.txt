# TEXT TO SPEECH CONVERTER


## Description
This is a simple and responsive Text to Speech (TTS) Converter built using HTML, CSS, and JavaScript. 
It uses the Web Speech API to convert the written text into spoken words using different voice options.

## Features

    - Convert text into speech with a single click
    - Choose from available system/browser voices
    - Clean and colorful gradient-based UI
    - Fully responsive for desktop, tablet, and mobile devices

## Technologies Used

    - HTML5
    - CSS3
    - JavaScript (Web Speech API)

##How to Use

    1. Open the project folder.
    2. Make sure the following structure is present:

        /project-folder
        ├── index.html
        ├── style.css
        ├── app.js
        └── images
             ├── play.png
             └── dropdown.png

    3. Open `index.html` in a web browser (Chrome recommended).
    4. Enter text in the textarea.
    5. Choose a voice from the dropdown.
    6. Click the "Listen" button to hear the speech.

Notes:
------
- The project uses the browser's built-in Speech Synthesis voices.
- Voices may vary depending on the browser and operating system.
- Requires internet access for loading some voice data (especially on first use).

## How to Run
No setup required. Simply download or clone the repository and open `index.html` in your web browser.
