# SIMPLE QUIZ APP

## Description
This is a simple and responsive Quiz App built using HTML, CSS, and JavaScript.  
It allows users to test their knowledge by answering multiple-choice questions.  
The app provides immediate feedback on selected answers and shows the final score at the end.  
It is optimized for both desktop and mobile devices with a clean, colorful UI.

## Features

    - Multiple-choice quiz with four options per question
    - Visual feedback for correct and incorrect answers
    - Score display at the end of the quiz
    - Option to restart the quiz
    - Smooth button hover effects and transitions
    - Fully responsive design for desktop, tablet, and mobile screens

## Technologies Used

    - HTML5
    - CSS3
    - JavaScript (DOM manipulation and event handling)

## How to Use

    1. Open the project folder.
    2. Ensure the following structure is present:

        /quiz-app
        ├── index.html
        ├── style.css
        └── app.js

    3. Open `index.html` in any modern web browser.
    4. Read the question and click on one of the answer buttons.
    5. The app will show whether the selected answer is correct or not.
    6. Click the "Next" button to proceed to the next question.
    7. After all questions are answered, your score will be displayed.
    8. Click "Play Again" to restart the quiz.

Notes:
------
- Green background indicates the correct answer.
- Red background indicates an incorrect selection.
- Hover color for buttons is distinct to avoid confusion with result colors.
- Layout and text automatically adjust for smaller screen sizes.

## How to Run
No setup required. Simply download or clone the repository and open `index.html` in your web browser.
